# ЗАДАНИЕ: Разбиение суммы на разряды по степеням числа

[Рекурсия](https://metanit.com/sharp/tutorial/2.11.php)
[Локальные функции](https://metanit.com/sharp/tutorial/2.20.php)

## Контекст

В финансовых и вычислительных приложениях часто нужно разложить число на составляющие части по степеням некоторого
базового числа. Например, разложение суммы на купюры: 1000 = 500 + 200 + 200 + 100 или разложение числа на степени
двойки: 23 = 16 + 4 + 2 + 1. Это позволяет оптимизировать платежи, распределять ресурсы по блокам, проводить
вычислительные разбиения.

## Цель программы

Написать консольную программу, которая:

1. Принимает с консоли целое положительное число N.
2. Принимает базу B (>1), например 2, 3, 5.
3. Рекурсивно разбивает число N на сумму степеней числа B (в порядке убывания).
4. Выводит:
    - список использованных степеней,
    - их количество,
    - итоговую сумму для проверки.

## Пример работы

Ввод: N = 23, B = 2  
Вывод: Разложение на степени числа 2: 16 + 4 + 2 + 1, Количество элементов: 4, Итоговая сумма: 23

Ввод: N = 100, B = 5  
Вывод: Разложение на степени числа 5: 25 + 25 + 25 + 25, Количество элементов: 4, Итоговая сумма: 100

## Обязательные требования

1. Рекурсивный метод должен выполнять разложение числа на степени базы B.
2. Итоговое значение возвращается через return.
3. Можно использовать локальную функцию внутри Main.
4. Использовать массивы для хранения результата допустимо.
5. Ввод с консоли — обычный Console.ReadLine().

## Ограничения

- Нельзя использовать классы и структуры.
- Нельзя использовать коллекции (List, Dictionary).
- Только базовые типы (int, long, bool) и массивы.
- Использовать только методы главы 2: арифметика, массивы, рекурсия, методы, return, локальные функции.

## Цель задания

- Практика рекурсии с реально полезным смыслом, а не искусственной.
- Отработка логики разбиения числа на части.
- Чёткая архитектура: метод для разложения, метод для вывода.
- Контроль результата через return, массивы и простые типы.
