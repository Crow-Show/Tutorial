# Задание: Учёт рабочего времени сотрудника

## Описание

Создайте консольную программу для учёта рабочего времени сотрудника за месяц. Программа должна:

### 1. Запрашивать у пользователя:
    - Имя сотрудника.
    - Тип рабочего графика: "Пятидневный" или "Сменный".
    - При необходимости, корректировки рабочего времени (штрафы или переработки) по отдельным дням.


### 2. Формировать месячный рабочий график:
    - Для "Пятидневного" графика: 22 рабочих дня по 8 часов.
    - Для "Сменного" графика: 15 рабочих дней по 12 часов.


### 3. Позволять вносить корректировки: 
```
- Штрафы уменьшают количество рабочих часов в день.
- Переработки увеличивают количество рабочих часов в день.
- Корректировки должны проверяться на допустимость:
- Рабочие часы не могут стать меньше 0.
- Переработка не может превышать 6 часов в день.
```

### 4. Рассчитывать итоговые показатели:
    - Фактически отработанные часы за месяц.
    - Баланс часов: разница между фактически отработанными часами и месячной нормой.
    - Статус месяца: долг, переработка или закрыт.


### 5. Формировать и выводить на экран итоговую таблицу с подробной статистикой:
    - Строка с номерами дней.
    - Строка с отработанными часами.
    - Строка с штрафами.
    - Строка с переработками.
    - Разделители между строками для наглядности.


### 6. Выводить итоговые данные сотрудника:
    - Имя сотрудника.
    - Тип графика.
    - Рабочие часы в день.
    - Месячная норма часов.
    - Фактические часы.
    - Количество внесённых корректировок.
    - Статус месяца.
    - Таблицу с детализированными часами по дням.


## Дополнительно

- Программа должна корректно обрабатывать неверный ввод и предлагать повторить ввод.
- Все расчёты и вывод должны быть выполнены в консольном приложении.
