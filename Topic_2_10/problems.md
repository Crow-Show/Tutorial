# Практическая задача: Управление доступом пользователя

## Описание
Разработай консольную программу, которая определяет доступ пользователя к различным ресурсам системы на основе набора вводимых данных.

Программа должна:

1. Считать с консоли основные сведения о пользователе:
    - Email
    - Пароль
    - Статус блокировки аккаунта
    - Подписка
    - VIP-доступ
    - Админский доступ
    - Наличие ключа

2. На основе введённых данных вычислять **базовый доступ** пользователя к системе (например, к сайту или основной панели).

3. Использовать базовый доступ как основу для вычисления **дополнительных прав и ресурсов**.

4. Определять и выводить в консоль состояние доступа пользователя к разным ресурсам, например:
    - Основной доступ на сайт
    - Премиум-контент
    - Админская панель
    - Тестовые функции
    - Конфиденциальные данные

## Требования

- Не использовать условные операторы (`if`, `?:`) или null-оператор (`??`) для вычисления доступа.
- Использовать только логические выражения и булевы переменные.

## Цель
Научиться моделировать систему прав доступа, выстраивая зависимости между базовыми свойствами пользователя и конкретными ресурсами, используя логические операции.
