# [**Операции присваивания**](http://metanit.com/sharp/tutorial/2.23.php)

Операции присваивания \- это придание чему-то какого-то значения. Обычно это придание значения переменной. Стандартная операция присваивания это: **переменная \= значение.**

Однако к стандартной операции можно добавить любой арифметический или поразрядный оператор, который доступен для этого типа данных в формате **переменная \+= значение.** В таком случае вместо **присвоения значения** переменной, будет проведена **соответствующая операция** с **текущим** значением.

**Переменная \+= значение** это **переменная \= переменная \+ значение**. К текущему значению переменной будет прибавлено то, что значение, что стоит после **\=.**     
```
int x = 0;  
x += 5 		// x = 0 + 5; x = 5
```
Операции присваивания имеют самый **низкий приоритет** среди всех операций и выполняются последними. Они выполняются строго **справа налево**. А также могут идти **группами**:     
```
int x = 0;  
int y = 2;  
int z -= y -= x += 10 / 2;
```
Будет выполняться в порядке:        
```
x = 0 + 10 / 2			// x = 5
y = 2 - (x += 10 / 2)		// y = -3   
z = 0 - (y - (x += 10 / 2))	// z = 3
```
