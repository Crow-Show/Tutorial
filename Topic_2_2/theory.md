# **[Переменные и константы](https://metanit.com/sharp/tutorial/2.25.php)**

Переменные и константы в C\# \- зарезервированные области памяти. Каждая принадлежит определенному типу и может хранить определенные данные, например целые числа, строки и значения, типа true или false. По сути это локальное именованное хранилище информации определенного типа. Более подробно:

**Переменные** \- изменяемая область памяти. Значение можно задать как прямо, так и через другую переменную, а также математическую операцию и ввод пользователя. Их значение может меняться и вычисляться уже после объявления.

Их можно объявлять без значения. В таком случае они будут не инициализированы. Далее возможно 2 варианта:

	**Локальная переменная** \- ошибка компиляции при выводе. Компилятор не может выводить и использовать неинициализированные переменные. А автоматической инициализации не происходит.

	**Поле класса/структура** \- в таком случае переменная автоматически инициализируется в условном значении “выключено”. Её значение будет null, 0, false  в зависимости от типа.

Однако есть модификатор, меняющий ситуацию. При объявлении можно задать значение **null** любой переменной вручную при помощи **?**. Это будет выглядеть так: **int?**. Такая переменная будет иметь значение **null**, если её не объявить, даже если этот тип переменные не подразумевает значение **null, как int**, который хранит только целые числа. Также для любой переменной можно задать значение **по умолчанию**. Оно будет использоваться, если **переменная \= null**. Сделать это можно оператором **??**.

Таким образом варианты объявления переменных без значения выглядят примерно так(для примера используется локальная):
```
int *age;* 		// нет значения. ошибка  
int? *age:*		// значение null. Ошибки нет. Выводит пустоту.  
int *age ?? 0;*		// ошибка. ?? работает только со значением null, int не может быть null   
int? *age ?? 0;*		// значение 0\. Переменной присвоено значение null принудительно
```
**Константы** \- по сути те же переменные. Но их значение неизменно после объявления. Существует 2 вида констант:

1. **const** \- классический вариант константы. Требует присвоения значения сразу при объявлении. Значение не может быть присвоено при компиляции. **Может** быть объявлена значением значением другой константы или математической операцией. Но **не может** быть объявлена переменной, поскольку значение переменной измениться может \- а константы нет. **const** может быть как локальной, так и часть класс/структуры.

2. **readonly** \- более гибкий вариант. Значение **может** быть задано во время компиляции программы, поэтому может быть объявлена более гибко. Подразумевает объявление переменной или вводом пользователя. Но **после** объявления значение изменить **уже нельзя**. Не может быть локальной. Всегда является частью класса/структуры.
