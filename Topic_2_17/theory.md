# [Switch](https://metanit.com/sharp/tutorial/3.45.php)
## Синтаксис
```
switch(выражение)
    case условие 1:                 //проверяет условие и выполняет код
        код
        break;                      //завершает работу switch после выполнения кейса
    case условие 2:
        код
        goto case условие 4        //если срабатывает кейс, вместо завершения переводит на указанный кейс, минуя остальные
    case условие 3:
        код;                        //без break позже после выполнения условия продолжается проверка кейсов по списку
    case условие 4:
        код
        break;                      //сюда переводит goto case кейса 2, минус кейс 3
    default                         //аналог else, выполняет действие, если кейсы не сработали
        код
        break;
```
## Синтаксис return

Возвращение значения из метода стандартным switch:
```
int метод()
{
switch (выражение)
    case условие:
        return значение;
}
```
Возвращение через укороченное присвоение значения переменной:
```
int метод()
{
int result or switch
    {
        условие 1 => значение 1
        уcловие 2 => значение 2
        _ => дефолтное значение
    };

return result;
}
```
Возвращение через укороченное присвоение значения return:
```
int метод()
{
return or switch
    {
        условие 1 => значение 1
        уcловие 2 => значение 2
        _ => дефолтное значение
    };
}
```
Возвращение напрямую через привязку switch к методу
```
int метод() => or switch
{
    условие 1 => значение 1
    уcловие 2 => значение 2
    _ => дефолтное значение
};
```
## Switch vs if
switch, как и if, используется для выбора ветки выполнения, но предназначен для работы с одним значением, которое
принимает ограниченный набор вариантов.
В отличие от if, switch не подходит для сложных условий и логических выражений, зато хорошо читается при работе с
состояниями, командами и enum.
Обычно switch используется как точка выбора сценария: внутри кейсов выполняются простые действия или вызываются методы,
а сложная логика выносится за его пределы.

# [Enum](https://metanit.com/sharp/tutorial/2.12.php)

Enum — это тип данных, представляющий строго ограниченный набор допустимых именованных значений одной сущности. Он
обеспечивает безопасность, позволяя компилятору гарантировать, что в переменной могут находиться только значения,
определённые в enum.

## Синтаксис
```
enum Название_перечисления
{                   //вариант, специально зарезервированный под неизвестные значения, не поддерживаемые этип типом
вариант 1,
вариант 2,
вариант 3
}
```

## Значения
Константы имеют целочисленные значения. По умолчанию от 0 до последнего сверху вниз.
Значения можно задать вручную в произвольном порядке. Также задать значение другим значением типа `Вариант 2 = Вариант 1`.

Обычно оставляют значения по умолчанию. Вручную задают значения только при необходимости, следуя определенной логике,
например внешний протокол, битовые флаги или совместимость.

## Практическое применение
enum используется для перевода "магических" значений в строго ограниченный, контролируемый и масштабируемый набор вариантов.

### Интерпретация
"Магические" значения обычно переводятся в enum значения при помощи интерпретатора - отдельного метода, возвращающего тип enum.
```
НазваниеEnum метод (тип "магическое" значение)
{
switch ("магическое" значение)
    {
    case вариант значения A:
        return НазваниеEnum.A;
    case вариант значения B:
        return НазваниеEnum.B;
    case вариант значения C:
        return НазваниеEnum.C;
    default:
        return НазваниеEnum.None;                   //если в enum нет подходящего варианта, возвращаем None
    }
}

enum НазваниеEnum
{
None,       //можно сделать вариант, специально зарезервированный под неизвестные значения, не поддерживаемые этип типом
A,
B,
C
}
```
