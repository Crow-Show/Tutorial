# Задание:
Создай программу, которая:

1. Выводит аккуратный блок текста с использованием интерполяции и вербатим строк:
```
-----------------------
Имя: <твое имя>
Возраст: <твой возраст>
Город: <твой город>
-----------------------
```
2. Каждое значение должно быть записано в переменные (string name, int age, string city) с корректным неймингом: camelCase для переменных.

3. Используй Console.WriteLine и интерполяцию $"..." для вставки значений.

4. Для оформления блока можно применить вербатим строку @, чтобы вывести рамку блока за один вызов Console.WriteLine.
